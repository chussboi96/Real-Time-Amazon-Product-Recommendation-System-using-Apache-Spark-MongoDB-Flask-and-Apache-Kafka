Future Data-Scientists

In this project, we utilized Spark MLlib on the Amazon reviews dataset to build a recommendation engine.

The Amazon reviews dataset is a massive dataset containing millions of reviews from customers all around the world. The primary goal of this project was to develop a recommendation engine that could provide personalized recommendations to individual users based on their past behavior and preferences. To accomplish this, we used TensorFlow and Keras libraries. We built the recommendation system using collaborative filtering with deep learning-based matrix factorization, as it can capture the underlying structure of user-item interactions more effectively than traditional methods.

In conclusion, we successfully integrated the recommendation system with a Flask-based web application, providing users with an easy-to-use interface to receive personalized product recommendations. The application leverages MongoDB for storage, Kafka for messaging, and TensorFlow for generating recommendations using the trained model.
